# Thunder Client
![image](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/tizuy1k1jeez7uzogpym.png)

![](https://github.com/rangav/thunder-client-support/blob/master/images/thunder-client.gif?raw=true)

Thunder Client is Rest API Client Extension for Visual Studio Code, with simple and clean design. The extension makes api testing very easy with Scriptless GUI based testing. and created by [Ranga Vadhineni](https://twitter.com/ranga_vadhineni).

Thunder Client has many Main Features such as :

 - List item
 - Lightweight API Client
 - Simple & Clean UI
 - Collection & Environment Variables
 - Scriptless Testing
